## Aqui está a ideia do meu projeto:
> A ideia é um site que não seja só um lugar com textos, mas também tenha formas diferentes de aprender sobre o assunto, como um mini jogo e uma área onde as pessoas possam informar problemas ambientais.

## O que terá?

- Explicação sobre o que são as mudanças climáticas.
- Um mini jogo para mostrar como nossas escolhas podem ajudar (ou prejudicar) o meio ambiente.
- Uma área para relatar problemas como queimadas, lixo e desmatamento.
- (Se der tempo) Um link para uma comunidade no Discord, onde o assunto será explicado de forma mais leve e rápida.
- (Se der tempo) uma integração com uma IA para ajudar a identificar problemas ambientais por imagens.

## Objetivo

Mostrar que qualquer pessoa pode aprender mais sobre o meio ambiente e que pequenas atitudes fazem diferença.
